GObject builder

Writing gobjects can be a tedious task. It requires a lot of typing and just doing a copy/paste requires a great deal of care. One obvious idea is to use some sort of templates for the class skeletons. and then run them through a special tool to generate the real C files. GOB/ (or GOB2) is such a tool. It is a preprocessor which can be used to build GObjects with inline C code so that there is no need to edit the generated C code. The syntax is inspired by Java and Yacc or Lex. The implementation is intentionally kept simple: the inline C code provided by the user is not parsed.
