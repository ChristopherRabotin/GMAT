================================= ura090_1.bsp =================================

While the file release form (below) is named "URA090," in fact this is a re-release of the
URA090 ephemeris. It reportedly made use of a few more data points for one satellite, but
more importantly, it used a revised (improved) step size during the integration process.
The authors report this solution is slightly better than the previous version, and it is
the one that has been published under the URA090 name.

For SPICE purposes, the SPK version of this re-release is named URA090_1.
