GMAT API Cheat Sheet¶

Loading the API¶

Python

MATLAB

import gmatpy as gmat

load_gmat

Asking for Help¶

Python

MATLAB

gmat.Help()

gmat.gmat.Help()

gmat.Help(<topic>)

gmat.gmat.Help(<topic>)

object.Help()

object.Help()

Examples

gmat.Help(“ScriptUsage”)

gmat.gmat.Help(“Objects”)

burn.Help()

burn.Help()

Object Creation¶

Python

MATLAB

obj = gmat.Construct(<type>,<name>)

obj = gmat.gmat.Construct(<type>, <name>)

Examples

Python

MATLAB

burn = gmat.Construct(“ImpulsiveBurn”, “Burn”)

burn = gmat.gmat.Construct( “ImpulsiveBurn”,”Burn”)
