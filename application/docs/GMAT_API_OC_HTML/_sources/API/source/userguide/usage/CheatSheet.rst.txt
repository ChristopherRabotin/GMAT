GMAT API Cheat Sheet
====================

Loading the API
---------------

+--------------------------------------+--------------------------------------+
| Python                               | MATLAB                               |
+======================================+======================================+
| import gmatpy as gmat                | load_gmat                            |
+--------------------------------------+--------------------------------------+


Asking for Help
---------------

+--------------------------------------+--------------------------------------+
| Python                               | MATLAB                               |
+======================================+======================================+
| gmat.Help()                          | gmat.gmat.Help()                     |
+--------------------------------------+--------------------------------------+
| gmat.Help(<topic>)                   | gmat.gmat.Help(<topic>)              |
+--------------------------------------+--------------------------------------+
| object.Help()                        | object.Help()                        |
+--------------------------------------+--------------------------------------+
| **Examples**                                                                |
+--------------------------------------+--------------------------------------+
| gmat.Help("ScriptUsage")             | gmat.gmat.Help("Objects")            |
+--------------------------------------+--------------------------------------+
| burn.Help()                          | burn.Help()                          | 
+--------------------------------------+--------------------------------------+

GMAT Objects
---------------

Listing Available Classes
*************************
+--------------------------------------+--------------------------------------+
| Python                               | MATLAB                               |
+======================================+======================================+
| gmat.ShowClasses()                   | gmat.gmat.ShowClasses()              |
+--------------------------------------+--------------------------------------+
| gmat.ShowClasses(<type>)               | gmat.gmat.ShowClasses(<type>)      |
+--------------------------------------+--------------------------------------+
| **Examples**                                                                |
+--------------------------------------+--------------------------------------+
| gmat.ShowClasses()                   | gmat.gmat.ShowClasses()              |
+--------------------------------------+--------------------------------------+
| gmat.ShowClasses("PhysicalModel")    | gmat.gmat.ShowClasses("Propagator")  |
+--------------------------------------+--------------------------------------+


Listing Created Objects
***********************

+======================================+======================================+

+--------------------------------------+--------------------------------------+
| gmat.ShowObjects(<type>)             | gmat.gmat.ShowObjects(<type>)        |

+--------------------------------------+--------------------------------------+


Object Creation
***************

+--------------------------------------+--------------------------------------+
| Python                               | MATLAB                               |
+======================================+======================================+
| obj = gmat.Construct(<type>,<name>)  | obj = gmat.gmat.Construct(<type>,    |
|                                      | <name>)                              |
+--------------------------------------+--------------------------------------+
| **Examples**                                                                |
+--------------------------------------+--------------------------------------+
| Python                               | MATLAB                               |
+--------------------------------------+--------------------------------------+
| burn =                               | burn =                               |
| gmat.Construct("ImpulsiveBurn",      | gmat.gmat.Construct(                 |
| "Burn")                              | "ImpulsiveBurn","Burn")              |
+--------------------------------------+--------------------------------------+


Object Field Access
*******************

+--------------------------------------+--------------------------------------+
| Python                               | MATLAB                               |
+======================================+======================================+
| value = obj.GetField(<FieldLabel>)   | value = obj.GetField(<FieldLabel>)   |
+--------------------------------------+--------------------------------------+
| obj.SetField(<FieldLabel>,<value>)   | obj.SetField(<FieldLabel>,<value>)   |
+--------------------------------------+--------------------------------------+
| obj.GetNumber(<FieldLabel>)          | obj.GetNumber(<FieldLabel>)          |
+--------------------------------------+--------------------------------------+
| **Examples**                                                                |
+--------------------------------------+--------------------------------------+
| V = burn.GetField("Element1")        | V = burn.GetField("Element1")        |
+--------------------------------------+--------------------------------------+
| burn.SetField("Element1",1.5)        | SetField("Element1",1.5)             |
+--------------------------------------+--------------------------------------+
| burn.SetField("Origin","Mars")       | SetField("Origin","Mars")            |
+--------------------------------------+--------------------------------------+
| V = burn.GetNumber("Element1")       | V = burn.GetNumber("Element1")       |
+--------------------------------------+--------------------------------------+

GMAT Script Access
------------------

Loading a GMAT script
*********************

+--------------------------------------+--------------------------------------+
| Python                               | MATLAB                               |
+======================================+======================================+
| gmat.LoadScript(<script>)            | gmat.gmat.LoadScript(<script>)       |
+--------------------------------------+--------------------------------------+
| **Examples**                                                                |
+--------------------------------------+--------------------------------------+
| gmat.LoadScript(                     | gmat.gmat.LoadScript(                |
| "../samples/Ex_GEOTransfer.script")  | "../samples/Ex_GEOTransfer.script")  |
+--------------------------------------+--------------------------------------+


Running a GMAT script
*********************

+--------------------------------------+--------------------------------------+
| Python                               | MATLAB                               |
+======================================+======================================+
| gmat.RunScript()                     | gmat.gmat.RunScript()                |
+--------------------------------------+--------------------------------------+

Saving a GMAT script

+--------------------------------------+--------------------------------------+

| "../scripts/New_GEOTransfer.script") | "../scripts/New_GEOTransfer.script") |
+--------------------------------------+--------------------------------------+
