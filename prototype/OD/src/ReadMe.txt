

This is a MATLAB prototype for OD functionality to be incorporated in GMAT.
You will need to download David Vallado's matlab astrodynamics utilities to
run this prototype.  Here is the url: http://celestrak.com/software/vallado-sw.asp

To install and test:
1)  Download all files located in the following relative path of the GMAT
svn repository located at sourceforge:  \branch\development\prototpye\OD\src
2)  Dowload Vallado's matlab astrodynamics utilities.  http://celestrak.com/software/vallado-sw.asp
3)  Unzip vallado's utilities and install.
4)  Add all folders in the OD\src directory and the vallado utilities to your path
5)  Type TestCase1 at the prompt
